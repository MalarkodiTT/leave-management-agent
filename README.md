# AI-Powered Leave Approval System

A simple, modern Leave Management System that leverages AI to process and evaluate leave requests automatically. Using the Groq API (Llama 3.3 model), the system analyzes the provided reason and provides an instant decision.

## 🚀 Features
* **Modern UI:** Clean, responsive interface built with Tailwind CSS.
* **AI Integration:** Uses the `llama-3.3-70b-versatile` model via Groq API for smart decision-making.
* **Structured Data:** The AI is prompted to return responses strictly in JSON format for easy parsing.
* **Easy Setup:** Lightweight architecture, perfect for students and developers.

## 🛠 Tech Stack
* **Frontend:** HTML, Tailwind CSS, JavaScript (Fetch API)
* **Backend:** Node.js
* **AI:** Groq API 
Live Link : https://leave-management-agent-bp9s.onrender.com/
